# Face Recognition Attendance System

A smart attendance system that automatically marks attendance using face recognition.

## Technologies
Python, Flask, OpenCV, NumPy, Pandas, HTML, CSS, JavaScript

## Features
- Face detection using OpenCV
- Face recognition using LBPH algorithm
- Automatic attendance recording
- Attendance stored in Excel files
- Lecturer login system
- Student attendance tracking

## How to Run

1. Clone the repository

git clone https://github.com/MatetiRithvika123-prog/face-recognition-attendance.git

2. Go to backend folder

cd backend

3. Create virtual environment

python -m venv venv
venv\Scripts\activate

4. Install dependencies

pip install -r requirements.txt

5. Run the project

python app.py

6. Open browser

http://127.0.0.1:5000
